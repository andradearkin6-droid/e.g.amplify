# Amplifi — Automating Employee Incentives in ABM

A working implementation of the platform described in the capstone demo
script: an admin panel for deploying marketing campaigns, an employee
portal for generating personal tracking links, a tracker/redirect engine
with simulated Account-Based Marketing (ABM) target detection and IP-based
fraud prevention, and an automated incentive ledger that dispatches
Slack/Teams webhooks when an employee crosses a points threshold.

Built with **plain Node.js only** — no `npm install` required.

## Run it

```bash
node server.js
